# Emory Women's Golf Website Overview

Hi!

I am **Zimo Li**, a student-athlete at Emory University. This is a concise overview of **Emory Women’s Golf Web 2.0**, a web application that tracks our team’s points game.

Starting in Fall 2025, our team introduced a point system where players earn points through practices, tournaments, and other team activities to unlock rewards.

## Impact

Before this project, all points were tracked manually on a whiteboard in the players' lounge, making updates slow and the system less transparent.

As the two computer science majors on the team, Katie and I thought it would be useful to build a digital version of the system, significantly improving efficiency and transparency. We developed the first version of the website during an Emory Hacks hackathon.

## Version 2.0

Building on the hackathon prototype, I redesigned and polished the website to improve:

- **Responsiveness** (adapts to different screen sizes)
- **Usability** (easy to use)
- **Overall user experience**

The website now works smoothly across both mobile and laptop devices.
